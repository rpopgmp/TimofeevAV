# TimofeevAV
# Методы анализа биомедицинских данных для выявления патологий сердечно-сосудистой системы

Выпускная квалификационная работа. Автоматическая классификация 12-канальных ЭКГ для выявления болезни Шагаса.

## Краткое описание

Реализован пайплайн обработки и классификации ЭКГ-сигналов из открытых наборов CODE-15%, SaMi-Trop и PTB-XL: шумоподавление, аугментация для компенсации дисбаланса классов, извлечение морфологических и статистических признаков, отбор признаков по VIF, а также обучение пяти моделей классификации — логистической регрессии, KNN, LightGBM, ResNet50 (1D) и Transformer.

Лучший результат показала ResNet50, объединяющая признаки сигнала и табличные признаки: **ROC-AUC 0,856, top-5% recall 0,343**.

## Воспроизведение

Эксперименты разрабатывались в Google Colab; пути в ноутбуках указывают на Google Drive (`/content/drive/MyDrive/Diplom2`). Для запуска локально необходимо изменить пути в первых ячейках каждого ноутбука.

Порядок запуска:

1. **`01_preprocess_data.ipynb`** — конвертирует исходные HDF5/EDF в формат WFDB, строит общую метадатафрейм, выполняет ресэмплинг до 400 Гц и шумоподавление, сохраняет `signals_raw.npy` и `meta_raw.csv`.
2. **`02_eda.ipynb`** — разведочный анализ распределений по наборам, проверка корректности предобработки.
3. **`03_process_data.ipynb`** — строит train/val/test сплиты, применяет аугментацию (только к train), нормализует сигналы, выделяет 351 признак, отбирает признаки по VIF (порог 50), сохраняет `meta_norm.csv`, `signals_norm.npy`, `features.csv`.
4. **`04_classic_ml.ipynb`** — обучает LogReg, KNN и LightGBM. Подбор гиперпараметров KNN и LightGBM — через Optuna по ROC-AUC.
5. **`05_train_resnet50.ipynb`** — обучает ResNet50 в двух режимах: только сигнал и сигнал + табличные признаки.
6. **`06_train_transformer.ipynb`** — то же для ECGTransformer.
